Executar o comando Dig

A Dig é uma ferramenta de rede utilizada para consultar os servidores DNS, veja o passo a passo conforme a plataforma que você está utilizado:

  • 1. Windows
  • 2. Mac OS X
  • 3. Linux

Windows

Instalar a ferramenta Dig

  1. Download a instalação estável do BIND do site http://www.isc.org/downloads/.
  2. Extraia todo o conteúdo do ZIP em uma pasta exclusiva.
  3. Execute o arquivo BINDInstall.exe.
  4. A instalação do BIND solicita uma conta de usuário. Informe a conta com privilégios restritos.
  5. Utilize a opção Tools Only para instalar somente as ferramentas Dig, host, nslookup e nsupdate.

Perform a Dig

  1. Abra a caixa de comando do Windows (+ R).
  2. Digite o comando cmd.
  3. Clique no botão OK para visualizar a tela de emulação do DOS.
  4. No prompt de comando, digite dig <hostname> +trace.
  5. Tecle Enter para visualizar o resultado do dig.

Mac OS X

  1. Abra a caixa de comando do Mac OS X (command + space).
  2. Digite terminal.
  3. Tecle Enter para visualizar o terminal do Mac OS X.
  4. No prompt de comando, digite dig <hostname> +trace.
  5. Tecle Enter para visualizar o resultado do dig.

Linux

Instalar a ferramenta Dig

O comando para a instalação da ferramenta Dig no Linux depende da versão que você está utilizando, abaixo o comando de instalação nas principais versões do Linux:

  • Red Hat/CentOS: sudo yum install bind-utils
  • Ubuntu/Debian: sudo apt-get update -y
{% highlight console %} #sudo apt-get install dnsutils -y {% endhighlight %}

Perform a Dig

  1. Abra o terminal do Linux.
  2. No prompt de comando, digite dig <hostname> +trace.
  3. Tecle Enter para visualizar o resultado do dig.
