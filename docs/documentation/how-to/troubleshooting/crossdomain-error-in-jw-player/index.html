Erro de crossdomain no JW Player

Problema

Ao tentar executar o live streaming, o player apresentar a mensagem “Cannot load M3U8 crossdomain access denied”.

Causa

Cadastro do CNAME não foi feito no Real-Time Manager.

Ações

Cadastre o CNAME no Real-Time Manager;

Inclua o mesmo CNAME no seu DNS;

Caso a realização das ações acima não resolvam o problema, abra um ticket de ocorrencia e informe a URL que está com problemas de acesso.

Não encontrou o que procurava? Abra um ticket.
