Perímetro de segurança programável para tráfego de entrada e saída na edge da rede.

O Network Layer Protection da Azion permite criar listas com base na rede, localização do usuário ou ASN e utilizar listas automáticas, como a rede Tor. Com isso, você pode bloquear, monitorar comportamentos suspeitos.

Exemplo de definição de preço

Se suas aplicações receberam, em 1 mês, 3 bilhões de requests provenientes dos Estados Unidos, o valor será calculado da seguinte forma:

O primeiro bilhão de requests será tarifado ao preço de 0,005 USD para cada 10.000 requests. Portanto, (1.000.000.000 / 10.000) x 0,005 USD = 500 USD.

Os próximos 2 bilhões de requests serão tarifadas pelo preço de 0,004 USD. Assim, (2.000.000.000 / 10.000) x 0,004 USD = 800 USD.

O valor total a ser pago no mês será de 1.300 USD, de acordo com a simulação.
