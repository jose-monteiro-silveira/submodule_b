Exemplo de definição de preço

Se sua aplicação com Application Acceleration habilitado gerou dentro de um mês 30 TB de dados trafegados no Brasil, o valor será calculado da seguinte forma:

Os primeiros 10 TB trafegados são calculados pelo preço de 0,040 USD por GB, portanto 10.000 x 0,040 USD = 400 USD.

Os próximos 20 TB trafegados são calculados pelo preço de 0,035 USD por GB, portanto 20.000 x 0,035 USD = 700 USD.

Os primeiros 400 GB serão tarifados pelo preço unitário de 1,50 USD, ou seja, 400 x 1,50 USD = 600 USD.

Nesse exemplo, o valor no mês será de 1.100 USD.
