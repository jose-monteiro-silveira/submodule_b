Definição de preço On Demand

Você será cobrado conforme o uso do produto, pela soma dos valores apurados para as métricas Invocations e Computing Time:

  Estados Unidos Canadá Europa Brasil América Latina
Compute Time (por mil segundos) 0,049 USD 0,049 USD 0,049 USD 0,147 USD 0,265 USD
Invocations (por milhão) 0,060 USD 0,060 USD 0,060 USD 1,80 USD 3,24 USD

Atenção: todos os valores apresentados são exibidos em dólares americanos (USD).
A cobrança pelos serviços é realizada pela Azion Technologies LLC.

Exemplo de definição de preço

Se suas aplicações invocam, nos Estados Unidos, uma edge function 5 milhões de vezes em um mês, e cada invocação executa em 30 milissegundos, você será tarifado como segue.

Compute time: (5.000.000 de invocações x 0,030 segundos / 1.000) x 0,049 USD = 7,35 USD.

Invocations: (5.000.000 / 1 milhão) x 0,60 USD = 3 USD.

O total que será tarifado, nesse exemplo, é de 10,35 USD.
