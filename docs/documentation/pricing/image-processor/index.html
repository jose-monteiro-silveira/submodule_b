Image Processor

Crie sua conta

Processamento de imagens just-in-time para otimizar a experiência dos usuários em seu Edge Applications.

O Image Processor automatiza o tratamento das imagens em seu workflow, permitindo redimensionar, recortar e aplicar filtros em grande escala para que você possa entregar a versão ideal da imagem para cada usuário.

Os preços da tabela abaixo são referentes ao modelo “Acordo On Demand”. Se você deseja contratar o formato “Acordo Reserva de Capacidade”, clique aqui para falar com nossa equipe.

Definição de preço On Demand

Você será cobrado conforme o uso do produto, pela métrica abaixo.

Images Processed

Este é o volume total de imagens processadas pelo serviço Image Processor, para otimização, corte, redimensionamento, aplicação de filtro e demais funcionalidades do serviço. A contabilização de imagens processadas ocorre a cada requisição dos Edge Servers da Azion para o serviço de Image Processor.

Exemplo de definição de preço

Se suas aplicações entregam, em um mês, 600 milhões de imagens nos Estados Unidos e o offload para seu conteúdo é de 95% ou seja, 95% de seu conteúdo é entregue pelo cache sem a necessidade de consultar a origem, serão processadas 30 milhões de imagens pelo Image Processor.

As primeiras 10 milhões de imagens processadas serão tarifadas a 1,60 USD para cada 10.000 imagens, ou seja, (10.000.000 / 10.000) x 1,60 USD = 1.600 USD.

As próximas 20 milhões de imagens processadas serão tarifadas a 1,26 USD para cada 10.000 imagens, ou seja, (20.000.000 / 10.000) x 1,28 USD = 2.560 USD.

O valor total a ser pago, nesta simulação, será de 4.160 USD.
