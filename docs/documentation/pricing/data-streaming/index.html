Os preços da tabela abaixo são referentes ao modelo “Acordo On Demand”. Se você deseja contratar o formato “Acordo Reserva de Capacidade”, clique aqui para falar com nossa equipe.

Definição de preço On Demand

Você será cobrado conforme o uso do produto, pela soma dos valores apurados para duas métricas: Data Streaming Requests e Data Streamed.

Data Streaming Requests

É o número total de eventos enviados pelo produto Data Streaming para seus endpoints, calculado conforme valores abaixo, para cada 10 mil requisições:

Data Streaming Requests Preço por 10.000 requisições
Primeiro 1 bilhão de requisiçōes / mês 0,024 USD
Próximos 4 bilhōes de requisiçōes / mês 0,010 USD
Próximos 15 bilhōes de requisiçōes / mês 0,004 USD
Acima de 20 bilhōes de requisiçōes / mês 0,002 USD

Data Streamed

É o volume total de dados trafegados através do produto Data Streaming, calculado conforme valores abaixo, por GB:

Data Streamed Preço por GB
Primeiros 10 TB / mês 0,085 USD
Próximos 40 TB / mês 0,080 USD
Próximos 100 TB / mês 0,060 USD
Próximos 350 TB / mês 0,040 USD
Próximos 500 TB / mês 0,030 USD
Próximos 4 PB / mês 0,025 USD
Acima de 5 PB / mês 0,020 USD

Atenção: todos os valores apresentados são exibidos em dólares americanos (USD).

Exemplo de definição de preço

Se a sua aplicação gerou, em 1 mês, 3 bilhões de eventos enviados pelo Data Streaming, consumindo 25 TB de dados, o valor será calculado da seguinte forma:

Data Streaming Requests

O primeiro 1 bilhão de requisições é calculado pelo preço de 0,024 USD para cada 10.000 requisições, portanto (1 bilhão / 10.000) x 0,024 USD = 2.400 USD.

Os próximos 2 bilhões de requisições são calculados pelo preço de 0,010 USD para cada 10.000 requisições, portanto (2 bilhões / 10.000) x 0,010 USD = 2.000 USD.

Neste exemplo, o valor no mês será de 4.400 USD.

Data Streamed

Os primeiros 10 TB serão calculados pelo preço de 0,085 USD por GB, portanto 10.000 GB x 0,085 USD = 850 USD.

Os próximos 15 TB serão calculados pelo preço de 0,080 USD por GB, portanto 15.000 GB x 0,080 USD = 1.200 USD.

Conforme o exemplo, o valor no mês será de 2.050 USD.

Valor total

O valor total a ser pago no mês será de 6.450 USD, de acordo com a simulação.
