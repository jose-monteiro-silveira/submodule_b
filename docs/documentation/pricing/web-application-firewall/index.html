O Azion WAF é um sistema de scoring que calcula uma pontuação para suas requisições e classifica as ameaças, permitindo o bloqueio com base na sensibilidade desejada.

Os preços da tabela abaixo são referentes ao modelo “Acordo On Demand”. Se você deseja contratar o formato “Acordo Reserva de Capacidade”, clique aqui para falar com nossa equipe.

Definição de preço On Demand

Você será cobrado conforme o uso do produto, pela soma dos valores apurados para as métricas Requests, Rule Sets e Rules:

Requests

É a soma total de requisições analisadas pelo WAF.

Rule Sets

Este é o número de configurações do WAF, ou conjuntos de regras, que podem ser compartilhadas entre diferentes domínios.

As primeiras 50 rule sets estão inclusas para todos os clientes. A partir desse volume, serão tarifados USD 200,00 por rule set.

Rules

Rules é o número de regras criadas pelo cliente, tanto para liberação (regras em whitelist) quanto para bloqueio (regras em blacklist).

As primeiras 200 rules estão inclusas para todos os clientes. A partir desse volume, serão tarifados USD 5,00 por rule.

Atenção: todos os valores apresentados são exibidos em dólares americanos (USD).

Exemplo de definição de preço

Se suas aplicações receberam, em 1 mês, 3 bilhões de requests provenientes dos Estados Unidos, e suas configurações de WAF se encontram dentro dos limites de rule sets e rules, o valor será calculado da seguinte forma:

O primeiro bilhão de requests será tarifado ao preço de 0,025 USD para cada 10.000 requests. Portanto, (1.000.000.000 / 10.000) x 0,025 USD = 2.500 USD.

Os próximos 2 bilhões de requests serão tarifadas pelo preço de 0,024 USD. Assim, (2.000.000.000 / 10.000) x 0,024 USD = 4.800 USD.

O valor total a ser pago no mês será de 7.300 USD, de acordo com a simulação.
