Support

Abrindo um ticket

Utilize os tickets para enviar questionamentos, sugestões ou problema que tenha ocorrido na utilização de nossos produtos.

Cada ticket possui um fluxo de atendimento, conforme o tipo, o nível de prioridade e o SLA contratado.

Para abrir um ticket acesse o Real-Time Manager com a sua conta e clique no item Support.

Na tela Tickets, clique no item New support ticket e preencha os campos abaixo conforme a sua necessidade:

Campo Definição
Subject Informe o título que identifique o seu ticket
Severity General Guidance

System Impaired

Production System Impaired

Production System Down

Business-critical System Down
Description Descreva a sua necessidade

Descrição dos níveis de Severidade

Nível Demanda
General Guidance Você tem uma dúvida ou deseja solicitar uma funcionalidade.
System Impaired Funções não críticas de sua aplicação estão se comportando de forma anormal ou você tem alguma dúvida urgente sobre desenvolvimento utilizando produtos Azion.
Production System Impaired Funções importantes para a sua aplicação estão comprometidas ou degradadas.
Production System Down Você não tem como contornar o problema e seu negócio está significativamente impactado. Funções importantes de sua aplicação estão indisponíveis.
Business-critical System Down Você não tem como contornar o problema e seu negócio está em risco. Funções críticas de sua aplicação estão indisponíveis.

Clique no botão Submit

Sempre que você tiver algum problema, por favor, abra um um ticket em nossa ferramenta de tickets, para agilizar a nossa tratativa necessitamos que seja enviado as seguintes informações:

  • Qual URL da sua configuração com a Azion que está tendo problemas?
  • Qual o comportamento esperado? O que não está de acordo?
  • Em que condições o problema acontece? (Browser, aplicativo, sistema operacional, operadora de internet, tipo de conexão com a internet)
  • Qual o passo-a-passo para reproduzirmos o problema?
  • A partir do dispositivo que está tendo problemas, acesse a URL http://netinfo.azion.com e colete o valor que aparece ao lado do campo UUID.

Acompanhando Tickets Abertos

Para acompanhar os tickets que você abriu em nossa ferramenta, acesse o Real Time Manager com a sua conta e clique no item Support.

Na tela Tickets, clique no item Check ticket status. Na primeira vez que acessar será exibido os tickets Open os Pendig, você pode clicar em All tickets para verificar todos os tickets.
